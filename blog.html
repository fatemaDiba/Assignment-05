Important FAQ

What is the Document Object Model (DOM)?

The DOM is a tree-like structure of a webpage that lets developers manipulate its elements, content, and styles using JavaScript.

How do you select an element from the DOM? in short

You can select an element from the DOM using JavaScript methods like: document.getElementById('id'), document.getElementsByClassName('class'), document.getElementsByTagName('tag'), document.querySelector('selector').

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element to handle events from its child elements. Itâ€™s useful because it reduces the number of event listeners, improving performance and handling dynamically added elements efficiently.

How do you manipulate an element's attributes and styles using the DOM?

Attributes: Use element.setAttribute('attr', 'value') to set an attribute and element.getAttribute('attr') to get its value.

Styles: Modify styles with element.style.property = 'value'. For example, element.style.color = 'red'.
